При использовании литерального синтаксиса {...} мы сразу можем поместить в объект несколько свойств в виде пар «ключ: значение»:

let user = {     // объект
  name: "John",  // под ключом "name" хранится значение "John"
  age: 30        // под ключом "age" хранится значение 30
};

// получаем свойства объекта:
alert( user.name ); // John
alert( user.age ); // 30
Значение может быть любого типа. Давайте добавим свойство с логическим значением:

user.isAdmin = true;


Для удаления свойства мы можем использовать оператор delete:

delete user.age;

Объект, объявленный через const, может быть изменён.

Например:

const user = {
  name: "John"
};

user.name = "Pete"; // (*)

alert(user.name); // Pete

||||

                                           Циклы While & For
Зачастую встаёт задача сделать одно и тоже действие много раз. Например: вывести товары из списка одним
за другим или перебрать все числа от 1 до 10 и для каждого выполнить одинаковый код (например сделать счётчик).
Цикл while:
1) Цикл while проверяет блок кода, пока указанное условие истино
2) Сначала проверяется условие для выполнения кода
3) Если условие истина, то блок кода выполняется
4) После выполнения блока кода, условие снова проверяется и если условие истина, то блок кода выполнятеся
и так до того как условие станет ложным

let i = 0
while ( i<5){
    alert(i)
    i = i + 1 || i++
} 

let i = 0
do{
    alert(i)
    i = i + 1 || i++
} while(i<5)

                                                                    Тема 9
                                                                Прерывание цикла
                                                               (Break & continue)
Обычно цикл завершается при вычисления условия значения false, но мы можем выйти из цикла в любой момент с помощью оператора break.
Пример: программа подсчитывает сумму вводимых чисел, до тех пор, пока пользователь их вводит с клавиатуры

let sum = 0;
while (true){
    let value = + prompt('Введите число: ')
    if (!value) break;
    sum = sum + value
}
alert(sum)

Оператор break полностью прекращает выполнение цикла и передаёт управление на блок кода за его телом.

Continue
При выполнении оператора, цикл не прерывается, а переходит к следующей итерации (если условие true). Оператор используют, если понятно, что на текущем повторе нечего делать.

for ( i = 0; i<10;i++ ){
    if ( i % 2 === 0) continue;
    alert(i);
}

                                                                     Тема 10
                                                             Конструкция switch и case default
Конструкция используется для выполнения различных дейтсвий в зависимости от значения выржения. Она предствляет собой альтернативу цепочки if else и обычно используется когда
нужно сравнивать одно значение со множеством возможных вариантов

С клавиатуры вводится число, если число равно 3, то выводим "мало", если число равно 4, то выводим "в точку", если число равно 5, то выводим "много", если число !==3, 4, 5, то 
выводим "мимо"

let a = +prompt("Угадай число: ");
switch (a) {
    case 3: 
    case 5: 
        alert('Рядом')
        break;
    case 4:
        alert('В точку')
        break;
    default: 
    alert('Мимо')
    break;
}

С клавиатуры вводится число от 0 до 3 включительно, с помощью конструкции switch и case выведите результат. Если число равно 0 то программа выведет "Вы ввели 0",
если число равно 1 то программа выведет "Вы ввели 1", если число равно 2 или 3 то программа выведет "Вы ввели 2 или 3", при вводе других чисел выведет "Число вне диапазона"
